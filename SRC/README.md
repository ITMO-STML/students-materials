
## Запуск
Сначала откройте файл конфигурацию корпуса, который хотите использовать, по пути `configs/benchmark`.
В поле `dataset_dir` укажите путь до файла корпуса, который должен использовать для оценки.

Затем откройте `eval.py` в корне проекта.
В кортех `BENCH, DATA` у нужного корпуса впишите путь до файла с перечнем файлов для нужного набора.
Затем выберите с нужным индексом.
Следом выберите одну из трёх моделей для оценки расскоментировав нужный вариант с переменной `MODEL`.
Запустите файл и ожидайте окончания его работы
