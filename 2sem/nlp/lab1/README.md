## Лабораторная работа № 1.1: "Извлечение признаков из текста"

### Задание:
1. объединить позитивный и негативный корпуса `RuTweetCorp`;
2. удалить стоп-слова, используя список стоп-слов из библиотеки `nltk`;
3. извлечь лингвистические признаки, используя тональный словарь [LinisCrowd 2015](1_lab/datasets/words_all_full_rating.csv) по следующему алгоритму:
    - токенизация;
    - лемматизация;
    - определение токенов, которые есть в тональном словаре и формирование списка средних значений тональности таких слов:
        ```
        # если список получился пустым, то записываем в него один элемент 0.
        было:
        ['первое',
        'успешно', - 2
        'радоваться', -1
        'как']

        стало:
        ['2', '1']
        ```
    - в качестве признаков использовать среднее, максимальное, минимальное, суммарное значение и количество положительных и отрицательных значений:
        ```
        [1.5, 2, 1, 3, 2, 0]
        ``` 
    - сохранить признаки в файл.
4. извлечь морфологические признаки с использованием  `pymorphy3`.
    - для каждого сообщения вычислить относительную частоту основных частей речи,
    - сохранить признаки в файл.
5. извлечь признаки с помощью метода BoW и `sklearn.feature_extraction.text.TfidfVectorizer`.
    - сократить размерность до 1000,
    - cохранить признаки в файл.

### доп. задание:
6. извлечь дополнительные морфологические признаки, [ссылка](http://opencorpora.org/dict.php?act=gram);  
7. извлечь лингвистические признаки, используя тональный словарь `LinisCrowd 2015` и `RuSentiLex`.

## Лабораторная работа № 1.2: "Извлечение признаков из текста"

### Задание:
Извлечь признаки из из корпуса RuTweetCorp;
1. `Word2vec`-признаки, обученные самостоятельно (!);
2. [предобученные](https://rusvectores.org/ru/models/) `Word2vec`-признаки
    > предобработка вида (токенизация, удаление пунктуации, лемматизация). одним из этапов преобработки является добавление частеречной разметки для каждого слова ([скрипт](https://github.com/akutuzov/webvectors/blob/master/preprocessing/rus_preprocessing_udpipe.py));
3. `Fasttext`-признаки, обученные самостоятельно;
4. предобученные `Fasttext`-признаки.

(!) модели `Word2vec` и `Fasttext` ставят в соответствие каждому слову вектор, но так как в сообщениях содержится различное количество слов, то для получения векторов одинаковой размерности для каждого сообщения необходимо произвести усреднение значений векторов всех слов.
