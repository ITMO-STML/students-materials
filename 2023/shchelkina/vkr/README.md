
# Нормализации текстов социальных сетей методами машинного обучения

Выполнила: Щелкина Оксана Темуриевна

Университет ИТМО, магистратура "Речевые технологии и машинное обучение", 2 курс, 2025 г.


## Назначение

Данный Jupyter Notebook предназначен для демонстрации анализа и нормализации текстов из социальных сетей на русском языке. Используются современные NLP-инструменты и предобученные модели для приведения неформального текста к литературной норме и оценки качества преобразования, а также производится дообучение моделей для решения данной залачи.


## Порядок работы

1. Установка и настройка зависимостей.
2. Импорт библиотек и подготовка среды.
3. Загрузка и анализ особенностей исходных текстов.
4. Загрузка корпуса нормализованных текстов.
5. Экспериментальный анализ существующих моделей нормализации текста
6. Расчёт метрик качества.
7. Дообучение моделей библиотеки Hugging Face для адаптации к задаче нормализации текста.
7. Сравнительный анализ результатов.


## Функциональные возможности

- Выделение особенностей текста: ссылки, эмодзи, упоминания, хэштеги, сленг, сокращения.
- Поддержка расчёта метрик: CER, WER, BLEU, METEOR, Accuracy, F1.
- Дообучение
- Демонстрация работы модели на реальных данных.


## Системные требования

### Требования к техническим средствам

- ОЗУ: от 8 ГБ (рекомендуется 16+ ГБ, в зависимости от модели и корпуса данных)
- GPU: (опционально) для ускорения вычислений

### Требования к программным средствам

- Python 3.9+
- Jupyter Notebook / Google Colab
- Необходимые библиотеки указанны в первых ячейках ноутбука.


## Запуск программы

### Авторизация

Не требуется при локальном запуске. В случае использования Google Colab возможно подключение Google Drive для загрузки файлов.

### Интерфейс системы

**Входные данные:**

- CSV-файл с колонками `original` и `normalized`, содержащий тексты в неформальном стиле, собранные из социальных сетей.

**Выходные данные:**

- Статистика по лингвистическим особенностям текстов
- Веса дообученных моделей и логи дообучения.
- Значения метрик качества нормализации

## Демонстрационный пример

**Пример входа: корпус данных в формате пар:**
```csv
original: "сег лан спс за встр"
normalized: "сегодня хорошо спасибо за встречу"
```

**Результат работы модели: дообученная модель, способная сгенерировать нормализованный текст, метрики качества данной модели**
```text
CER: 0.1538
WER: 0.3333
BLEU: 0.7452
METEOR: 0.8011
Accuracy: 0.8571
F1: 0.8235
```
