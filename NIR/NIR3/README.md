# WIEM-based Graph Completeness Estimation (Streamlit Demo)

## Описание

Данный интерфейс представляет собой интерактивное Streamlit-приложение для
**оценки структурной полноты графа знаний** на основе текста
с использованием **WIEM-подобной метрики (Weighted Incremental Edge Method)**.

Приложение реализует полный исследовательский пайплайн:


Интерфейс предназначен для демонстрации и анализа метода,
а также для отладки пайплайна до подключения реальной LLM.

---

## Теоретическая основа (WIEM)

Метрика основана на следующей идее:

> Полнота графа определяется долей **семантически обоснованных отношений,
> которые отсутствуют в графе**, но могут быть восстановлены из текста.

Формально используется WIEM-подобная оценка:

\[
\text{Completeness} = 1 - \frac{\sum w_{\text{missing}}}{\sum w_{\text{predicted}}}
\]

где:
- \( w \) — вес (confidence) отношения,
- *predicted* — отношения, поддерживаемые текстом,
- *missing* — подмножество predicted, отсутствующее в исходном графе.

---

## Основные возможности интерфейса

- загрузка примеров из датасета **WebNLG**
- отображение:
  - текста,
  - RDF-триплетов графа,
  - списка сущностей
- вычисление **WIEM-подобной метрики полноты**
- визуализация:
  - missing edges (пропущенные рёбра),
  - confidence для каждого ребра,
  - распределения полноты по нескольким примерам
- просмотр:
  - промпта, отправляемого модели,
  - JSON-ответа модели

## MOCK-режим 

По умолчанию интерфейс работает в **MOCK-режиме**.

В этом режиме:
- вместо реальной LLM используется детерминированный генератор,
- имитирующий структурированный JSON-ответ модели,
- архитектура пайплайна и формула WIEM **полностью сохранены**.

Это позволяет:
- демонстрировать интерфейс без запуска тяжёлой модели,
- отлаживать метрику и визуализацию,
- использовать приложение в средах без GPU.

Подключение реальной LLM (например, Qwen через vLLM) осуществляется
путём замены mock-функции на реальный вызов модели
без изменения остального кода.

## Установка и запуск (локально)

### 1. Установка зависимостей

```bash
pip install streamlit datasets pandas
```

### 2. Запуск программы

streamlit run Streamlit.py

После запуска интерфейс будет доступен по адресу: http://localhost:8501
