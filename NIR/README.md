## Оценка возраста человека по речевому сигналу с использованием методов глубокого машинного обучения  

Цель этой работы – анализ современных методов глубокого обучения для задачи оценки возраста человека по речевому сигналу и выявление возможных оптимизаций для улучшения уже существующих результатов в этой области, а также тестирование моделей с открытым исходным программным кодом для сверки результата и объективной оценки возможности улучшения. В качестве языка программирования используется python, так как именно он является основным при решении задач машинного обучения. В дальнейшем планируется разработать систему предсказания возраста по речевому сигналу, которая будет опережать по метриками точности существующие модели.
