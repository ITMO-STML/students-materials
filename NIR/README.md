## Описание проекта

Данный скрипт реализует детектирование дипфейков в аудиозаписях с использованием акустических признаков **eGeMAPS** (Geneva Minimal Acoustic Parameter Set) и классификатора **Random Forest**.

**Основные возможности:**
- Извлечение 88 акустических признаков из аудиофайлов
- Сравнение двух подходов к валидации:
  - Случайное разбиение выборки
  - Кросс-валидация по дикторам (Leave-One-Group-Out)
- Визуализация результатов

## Исходные данные

### Датасет

Проект использует датасет **In-The-Wild** для детектирования дипфейков.

> **Ссылка на данные:** [https://huggingface.co/datasets/mueller91/In-The-Wild](https://huggingface.co/datasets/mueller91/In-The-Wild)

### Структура данных

├── meta.csv # Файл метаданных
├── audio1.wav # Аудиофайлы
├── audio2.wav
└── ...

### Формат файла meta.csv

| Колонка | Описание | Значения |
|---------|----------|----------|
| `file` | Имя аудиофайла | Пример: "sample_001.wav" |
| `label` | Тип аудио | `bona-fide` (реальная) или `spoof` (дипфейк) |
| `speaker` | ID говорящего | Уникальный идентификатор |

### Требования к аудиофайлам

- Формат: WAV (рекомендуется)
- Частота дискретизации: 16 кГц или выше
- Тип: моно

## Результаты эксперимента

### Кросс-валидация по дикторам

| Метрика | Значение |
|---------|----------|
| **Speaker-wise CV accuracy** | 77.39% |
| **Drop vs random** | 19.50% |

### Детальный анализ признаков

| Признак | Значение |
|---------|----------|
| **Jitter** | 0.218 |
| **Shimmer** | 0.906 |
| **MFCC1** | 0.537 |

> **Примечание:** Jitter и Shimmer являются важными акустическими признаками, характеризующими нестабильность голоса. MFCC1 (первый мел-кепстральный коэффициент) отражает общий энергетический уровень сигнала.

### Интерпретация результатов

- **Speaker-wise CV accuracy (77.39%)** — показывает способность модели обобщаться на новых дикторах
- **Drop (19.50%)** — разница между случайной валидацией и валидацией по дикторам. Значение около 20% указывает на наличие некоторой зависимости от конкретных дикторов, но модель сохраняет приемлемую обобщающую способность
