РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
1 Назначение
Данное програмное обеспечение представляет собой систему на основе LLM, предназначенную для генерации тестового вывода на основе заданной темы. Система может использоваться в задачах генерации рефератов, эссе и прочих учебных материалов.
2 Порядок работы
Пользователь указывает тему работы, вводит API-ключ, выбирает модели для агентов. Затем система обрабатывает запрос и возвращает результат в виде текстового ответа и доступного для скачивания файла.
3 Функциональные возможности
•	Анализ темы и требований
•	Поиск и обработка источников по заданной теме
•	Генерация текста как ответ по запросу
4 Системные требования
4.1 Требования к техническим средствам
●	CPU: не менее 4 ядер
●	RAM: от 8 ГБ
●	GPU: NVIDIA с 4 ГБ видеопамяти и выше (для ускорения работы модели)
●	Диск: от 20 ГБ свободного пространства

4.2 Требования к программным средствам
Операционная система: Windows 10
Python 3.8.1
Ollama 0.9.2
Библиотеки:
•	requests
•	streamlit
•	python-docx
•	ollama 
5 Запуск программы
5.1 Авторизация
Авторизация не требуется. Перед генерацией текста необходимо ввести API-ключ сервиса Serp-API.
5.2 Интерфейс системы
Входные данные:
●	Текстовый запрос
●	Выбор из выпадающего списка
Выходные данные:
●	Ответ модели на текстовый запрос
●	.docx файл для скачивания
