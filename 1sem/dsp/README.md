### Лабораторная работа 1: Частотный конвертер


**Описание**

Программа для преобразования дискретного сигнала из одной частоты дискретизации в другую.


**Входные данные**

- Дискретный сигнал `x` с частотой дискретизации `N1` (Hz)
- Входной аудиофайл: `music.mp3`


**Задача**

Преобразовать сигнал `x` в сигнал `y` с частотой дискретизации `N2` (Hz).

### Лабораторная работа 2: Декодирование DTMF сигналов


**Описание**

Программа для декодирования DTMF (Dual-Tone Multi-Frequency) сигналов с использованием алгоритма Герцеля.


**Входные данные**

- Аудио сигнал `x` с частотой дискретизации `F_d = 44100 Hz`
- Входной аудиофайл: `DTMF_Lab15.wav`


**Задача**

Реализовать алгоритм Герцеля для декодирования DTMF сигнала и восстановления исходной последовательности символов.

### Лабораторная работа 3: Определение частот трезвучия в аудио сигнале

**Описание**

Программа для анализа аудиосигнала трезвучия с определением основных частот (P1, P2, P3) и тональности с использованием спектрального анализа и проверки обертонов.

**Входные данные**

- Аудио сигнал с частотой дискретизации `F_d = 16000 Hz`, 16-бит, моно, формат PCM WAV
- Входной аудиофайл: `DTLab16.wav`

**Задача**

Определить частоты основного тона (P1), терцового (P2) и квинтового (P3) тонов в заданном аудиосигнале, проверить гармоники и установить тональность трезвучия.

### Лабораторная работа 4: Восстановление аудиосигнала по амплитудной спектрограмме с помощью алгоритма Гриффина-Лима

**Описание**

Программа для восстановления временного сигнала из амплитудной спектрограммы с использованием итеративного алгоритма Гриффина-Лима.

**Входные данные**

- Аудио сигнал с частотой дискретизации `sr = 22050 Hz`
- Входной аудиофайл: `voice_message.ma4`

**Задача**

Реализовать алгоритм Гриффина-Лима для восстановления фазовой информации из амплитудной спектрограммы и получения синтезированного аудиосигнала, максимально приближенного к оригиналу.
