### Лабораторная работа 1: Частотный конвертер


**Описание**

Программа для преобразования дискретного сигнала из одной частоты дискретизации в другую.


**Входные данные**

- Дискретный сигнал `x` с частотой дискретизации `N1` (Hz)
- Входной аудиофайл: `music.mp3`


**Задача**

Преобразовать сигнал `x` в сигнал `y` с частотой дискретизации `N2` (Hz).

### Лабораторная работа 2: Декодирование DTMF сигналов


**Описание**

Программа для декодирования DTMF (Dual-Tone Multi-Frequency) сигналов с использованием алгоритма Герцеля.


**Входные данные**

- Аудио сигнал `x` с частотой дискретизации `F_d = 44100 Hz`
- Входной аудиофайл: `DTMF_Lab15.wav`


**Задача**

Реализовать алгоритм Герцеля для декодирования DTMF сигнала и восстановления исходной последовательности символов.
