# Лабораторные работы по NLP

## Описание
Содержатся Jupyter-ноутбуки с лабораторными по курсу "Обработка естественного языка".  

## используемый датасет
https://www.kaggle.com/datasets/evamars9/rutwittcorp

## Структура
- `lab1.ipynb` — Извлечение признаков из текста с помощью TF-IDF, Word2Vec, FastText, сентимент-словарей и анализа частей речи с последующей классификацией.
- `lab2.ipynb` — Классификация текстовых признаков с помощью различных нейронных сетей
- `lab3.ipynb` —  выявить корреляцию (или ее отсутствие) между размером словаря токенизатора и качеством модели (выраженном в f1) для задачи сентимент-анализа
