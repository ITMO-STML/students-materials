## Описание проекта
Реализация наивного байесовского классификатора для гауссовских данных "с нуля". 
Шаги:
- Генерация синтетических данных для классификации
- Реализация Gaussian Naive Bayes
- Визуализация границ решения
- Контейнеризация с помощью Docker

## Структура
- `model.py` - реализация наивного байеса
- `data_generation.py` - генерация данных
- `main.py` - основной скрипт с визуализацией
- `requirements.txt` - зависимости
