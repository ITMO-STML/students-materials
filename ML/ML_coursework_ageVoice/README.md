# Оценка возраста по речевому сигналу

## Описание задачи

В ходе исследования решалась задача оценки возраста по речевому сигналу с использованием глубокого обучения. 

**Цель работы** – определение архитектуры нейронной сети, повышающей F1-меру при классификации возрастных групп:

- Молодые люди (0-25 лет)
- Взрослые (26-50 лет)  
- Пожилые (старше 50 лет)

## Данные и признаки

Исследование проводилось на корпусе **TIMIT** (6300 записей, 630 дикторов). Для обучения были извлечены следующие типы признаков:

- **Классические признаки**: MFCC, мел-банки фильтров
- **Нейросетевые признаки**: wav2vec 2.0, WavLM

## Результаты

### Baseline
В качестве baseline была выбрана полносвязная сеть (FCNN), которая обучалась последовательно на каждом типе признаков. Лучший результат был достигнут на MFCC:

| Модель | Признаки | F1-score | Accuracy |
|--------|----------|----------|----------|
| FCNN (baseline) | MFCC | 0.7644 | - |

### Улучшенные архитектуры

Усложнение архитектуры до ResNet дало незначительный прирост:

| Модель | Признаки | F1-score |
|--------|----------|----------|
| ResNet | MFCC | 0.7761 |
| ResNet | WavLM | 0.7609 |

### Stacking Ensemble

Объединение всех признаков в вектор размерностью 2128 и построение stacking-ансамбля (ResNet + FCNN с мета-классификатором) позволило достичь наилучшего результата:

| Показатель | Значение |
|------------|----------|
| **F1-score** | **0.7790** |
| **Accuracy** | **0.7981** |

### Анализ по классам

Лучшее качество достигнуто для класса **26-50 лет** (F1 = 0.8677), что объясняется его преобладанием в корпусе данных. Два других класса распознаются хуже из-за несбалансированности TIMIT.

## Выводы

1. Полученный результат (F1 = 0.7790) превосходит baseline и значительно опережает результат бенчмарка Vox-Profile

2. Незначительный прирост при усложнении архитектуры объясняется тем, что нейросетевые признаки wav2vec и WavLM являются высокоуровневыми, поэтому возможности улучшения за счет усложнения классификатора ограничены

3. Дисбаланс классов в корпусе TIMIT препятствует получению более высоких метрик на редких возрастных группах
