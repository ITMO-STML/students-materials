# Суммаризация русскоязычных длинных документов на основе больших языковых моделей

Система для суммаризации русскоязычных длинных документов на основе большой языковой модели GigaChat.

Может использоваться как для получения краткого содержания в виде текста, так и для структурированной суммаризации конференций (в формате JSON).

## Структура репозитория

```
├── structured_summarization
│ ├── lib # Директория с модулями для обращения к GigaChat с учётом выставленных гиперпараметров модели
│ ├── longdocpipeline
│ │ ├── conf # Директория с промптами для модели и с конфигурационным файлом для модели GigaChat
│ │ │ ├── split_conference_prompts # Промпты для разделения конференций на доклады (в рамках структурированной суммаризации)
│ │ │ │ ├── type_identifier_system_prompt.txt # Системный промпт для разделения конференции на доклады
│ │ │ │ └── type_identifier_user_prompt.txt # Пользовательский промпт для разделения конференции на доклады
│ │ │ ├── application.yaml # Конфигурационный файл для GigaChat и системы суммаризации длинных документов
│ │ │ ├── default_prompts.json # Промпты для суммаризации длинных документов
│ │ │ └── logger.yaml # Конфигурационный файл для логгирования
│ │ ├── config # Директория с модулем для обработки конфигурационного файла модели GigaChat и системы суммаризации длинных документов
│ │ ├── pipeline # Пайплайн для обработки длинных документов
│ │ │ ├── conference_summarization # Директория с модулями для суммаризации длинных конференций
│ │ │ │ ├── conference_splitter.py # Модуль для разделения конференции на чанки с установленным количеством токенов, для исправления ошибок диаризации и для деления конференции на доклады 
│ │ │ │ ├── conference_summarizer.py # Модуль для суммаризации конференций
│ │ │ │ ├── diarization_corrector.py # Модуль для исправления ошибок диаризации
│ │ │ │ ├── input_text_preprocessor.py # Модуль для предобработки текста конференции
│ │ │ │ ├── schemas.py # Pydantic схемы для структурированной суммаризации конференций
│ │ │ │ └── talks_splitter.py # Модуль для разделения конференции на доклады
│ │ │ ├── graphs # Директория с графами для суммаризации длинных документов
│ │ │ │ ├── abstract.py # Абстракция для графов
│ │ │ │ ├── concat.py # Граф, конкатенирующий краткие содержания
│ │ │ │ ├── map_reduce.py # Граф Map-Reduce
│ │ │ │ └── refine.py # Граф Iterative Refinement
│ │ │ ├── constants.py # Константы для использования пайплайна
│ │ │ ├── gigachat_provider.py # Вызов модели GigaChat с установленными гиперпараметрами
│ │ │ ├── inference_manager.py # Основной модуль для обработки длинных документов с использованием графов
│ │ │ ├── postprocessing.py # Постобработка результата обработки длинных документов
│ │ │ ├── preprocessing.py # Предобработка текстов для их передачи в пайплайн обработки длинных документов
│ │ │ └── prompt_setup.py # Создание промптов (в нужном для langchain формате) для пайплайна обработки длинных документов
│ ├── resources # Примеры входных данных для тестирования системы
│ ├── test_conference_summarization # Модуль для тестирования структурированной суммаризации длинных документов
│ └── conference_summarization.ipynb # Блокнот с демонстрацией примера работы системы
```
