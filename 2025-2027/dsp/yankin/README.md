### Лабораторная работа 1: Частотный конвертер


**Описание**

Программа для преобразования дискретного сигнала из одной частоты дискретизации в другую.


**Входные данные**

- Дискретный сигнал `x` с частотой дискретизации `N1` (Hz)

- Входной аудиофайл: `music.mp3`


**Задача**

Преобразовать сигнал `x` в сигнал `y` с частотой дискретизации `N2` (Hz).
